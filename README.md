# Proje 7: C# Console Uygulaması ile ASP.NET Core API Verisi Kullanımı

Bu projede, ASP.NET Core kullanarak geliştirdiğimiz şehirler ve hava durumu verilerini içeren API'yi, C# Console uygulaması üzerinden nasıl kullanacağımızı gösteren bir uygulama geliştirdik. API'den alınan verileri, konsol uygulaması içinde işleyip kullanıcıya sunmayı amaçladık.

## İçerik

- **ASP.NET Core API**: Şehirler ve hava durumu verilerini sağlayan API.
- **C# Console Uygulaması**: API'den verileri çekip işleyen ve kullanıcıya sunan bir konsol uygulaması.

## Özellikler

- API'den JSON formatında hava durumu verisi çekme.
- C# Console uygulamasında JSON verisini işleyerek kullanıcılara dinamik bilgi sunma.
- Kullanıcı etkileşimi ile hava durumu verilerinin filtrelenmesi.
- Basit ve anlaşılır bir kullanıcı arayüzü.

## Kullanım

1. **API'yi Çalıştırma:**
   - İlk önce **Proje 6**'da geliştirdiğimiz ASP.NET Core API projesini çalıştırın.
   - API, şehirler ve hava durumu verilerini JSON formatında dönecektir.

2. **Console Uygulamasını Çalıştırma:**
   - **Proje 7**'yi bir C# Console uygulaması olarak açın.
   - Uygulama, API'ye istek göndererek şehirlerin hava durumu bilgilerini alacak ve kullanıcıya gösterecektir.

3. **Veri Görüntüleme:**
   - Uygulama, API'den alınan hava durumu verilerini konsol üzerinden dinamik bir şekilde gösterecek.

## Teknolojiler

- **ASP.NET Core**: API geliştirmek için.
- **C#**: Console uygulaması geliştirmek için.
- **JSON**: API ile veri alışverişi yapmak için.
